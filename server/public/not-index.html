Photos API test

This HTML/JS page tests the Equipment API by listing all; then creating, finding, modifying, and deleting a equipment resource using the REST API. A few notes:

  • API endpoints, property names and values for the example equipment app are hardcoded in the client-side /js/api.js file linked in this HTML page.
  • To test your own API you will have to modify these values.
  • If your application does not need to handle file uploads, you will not need to use this FORM's file input element in the HTML, or FormData in the JS, for your POST/create operation. Simple JSON will do.
  • See the console for output as the process completes.
